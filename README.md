# Stock Market Analysis Project 

## Goal
Analyze historical stock price data to identify price trends, daily returns, and volatility for evaluating stock performance and risk.

## Code
- [stock-market-analysis.py](stock-market-analysis.py)

## Description 
This project retrieves historical stock price data for Apple Inc. (AAPL) using the Yahoo Finance API. The dataset includes daily adjusted closing prices, which are cleaned and processed to compute financial metrics such as daily returns, moving averages, and annualized volatility. The analysis focuses on understanding price behavior and trend patterns over a certain period of time. 

## Technology
- Python, pandas, NumPy, Matplotlib, yfinance

## Skills
- Data Cleaning, Financial Data Analysis, Data Visualization, Preprocessing

## Results
The analysis calculated daily returns and 20 and 50-day moving averages to identify trends and assess short and long-term stock performance. Annualized volatility measured overall risk, and visualizations clearly highlighted price movements and patterns over time.
